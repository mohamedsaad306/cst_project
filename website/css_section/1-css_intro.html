Why use CSS?

  1. Managing Style at any sites.
  2. The Separation of Structure and Presentation.
  3. Ease of maintenance of site.
  4. Decreased production work hours

CSS Solved a Big Problem

HTML was NEVER intended to contain tags for formatting a web page! HTML was created to describe the content of a web page When tags like

<font> and color attributes were added to the HTML 3.2 specification, it started a nightmare for web developers. Development of large websites, where fonts and color information were added to every single page, became a long and expensive process. To solve this problem, the World Wide Web Consortium (W3C) created CSS.
