Table Borders

To specify table borders in CSS, use the border property. The example below specifies a black border for

<table>, <th> , and <td> elements:
Example
table, th, td { border: 1px solid black; }

Notice that the table in the example above has double borders. This is because both the table and the

<th> and <td> elements have separate borders.

Collapse Table Borders

The border-collapse property sets whether the table borders should be collapsed into a single border:

table { border-collapse: collapse; } table, th, td { border: 1px solid black; }

Table Width and Height

Width and height of a table are defined by the width and height properties. The example below sets the width of the table to 100%, and the height of the

<th> elements to 50px:
Example

table { width: 100%; } th { height: 50px; }

Table Color

The example below specifies the background color and text color of

<th> elements:

Example

th { background-color: #4CAF50; color: white; }
